About Tippet
This web application was designed to improve the fly fishing experience for everyone. Anything from knot tying to fish handling can be found here. No matter what your experience level is, you'll find something here that will help you continue learning. Are you a beginner that needs help getting started? We're here to help! Just head over to the Gear and Casting sections to start learning the basics. As an advanced user you can still find helpful content in this app, such as Fly Tying. No matter how much you read, though, nothing will beat quality time on the water. Head over to the Outfitters section to find great local fly shops that can set you up for success on your local waters. When you make it to your favorite spot, remember that Tippet is still there for you. This app has been designed so that you can view it on your mobile device while you're on the move.
[ADDITIONAL DETAILS - REMOVE AT LAUNCH]

The goal for this project was to get a head start on my senior project next semester. I wanted to create a fully responsive mobile-first web app that involves fly fishing, one of my personal hobbies. I will most likely use a MEAN stack to implement this. I wanted to focus on the frontend work for the NM231 final project. Some of the aesthetic work I've learned in this class puts me a step ahead. I've briefly experimented with W3.CSS previously, but this time I wanted to learn some Bootstrap. It has taken a considerable amount of time to find and read documentation on the new Bootstrap 4, but this will pay off for me soon. I tried experimenting with Mapbox and Leaflet with this project. After many test runs, I decided Google Maps would be more compatible when designing the backend later. After getting my API key from Google, it was just a matter of writing some Javascript to implement it. I plan to have a database later as well. It will store things like fly patterns, locations, and user's login information.
